# Nonguen-GE

El siguiente repositorio tiene como finalidad compartir el código de generación de reclasificación
de coberturas de suelo para el Parque Nacional Nonguén - Chile. El proceso se irá constantemente
actualizando a medida que se afinen los puntos para entrenar las firmas espectrales.

## Herramientas y métodos

El código estará escrito en *javascript* y se ejecutará en la consola que dispone *GEE*
